# Como usar o Firebase?
- Instalação:
	Comando: sudo npm install -g firebase-tools
	Fazer login com a conta Google: firebase login
	No diretório do projeto: firebase init

# App Universitário:

**DEFINIÇÃO DO PROJETO:**
O app **Agenda Universitária** busca unir organização, produtividade e tecnologia em um só ambiente, criando uma ferramenta que não apenas apoia a gestão acadêmica, mas também contribui para a formação de hábitos mais consistentes de estudo e aprendizado.
# Funcionabilidaddes do App:
- **Sistema de Login (mínimo: aluno)**
    
    - Mostra noção de autenticação.
        
    - Pode começar simples (usuário e senha armazenados localmente ou via Firebase).
        
- **Cadastro e listagem de disciplinas por semestre**
    
    - Estrutura organizada.
        
    - Permite expansão futura (como plano de aula e gráficos).
        
- **Agenda/Calendário integrado ao app**
    
    - Visualização semanal/mensal.
        
    - Associa eventos (provas, trabalhos, aulas).
        
- **Notificações**
    
    - Lembrete de prova, entrega de trabalho, horário de aula.
        
    - Agrega muito valor ao projeto.
        
- **Rotina de estudos**
    
    - O aluno monta sua própria agenda paralela (estudos extras).
        
    - Demonstra preocupação com a experiência do usuário.
	    
- **Plano de aula vinculado à disciplina**
    
    - Visualização dos conteúdos programados.
        
    - Mais útil para professores, mas agrega para o aluno também.
        
- **Chamada de turma (para professores)**
    
    - Boa para mostrar login em camadas (aluno vs. professor).
        
    - Difícil de implementar 100% sem backend robusto.
        
- **Gráficos de desempenho**
    
    - Notas, presença, horas de estudo.
        
    - Demonstra uso de bibliotecas externas e visão analítica.
        
- **Integração com Google Calendar**
    
    - Impressiona na apresentação.
        
    - Mostra conhecimento em APIs externas.
        
- **Integração com Google Classroom**
    
    - Muito interessante, mas é complexo (precisa autenticação OAuth e leitura de dados externos).
        
    - Melhor ser apresentado como **visão futura**.

# Sistemas e linguagens utilizadas:
- **Linguagem principa:** TypeScript
- **Frontend:** React Native(via Expo)
- **Backend:** Firebase
- **Gerenciamento de estado:** Context API ou REduxToolkit
- **Banco de Dados:** Firebase FIrestore
- **Autenticação:** Firebase Authentication
- **Notificações:** expo-notifications e Push notification(VIa Firebase CLoud Messaging)
- **Integrações:** Google Calendar API
- **Visualização de Dados:** Recharts
